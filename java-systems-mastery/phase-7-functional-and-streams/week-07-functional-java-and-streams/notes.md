## Week 07 - Functional Java and Streams (Interview Trade-off Stub)

Streams matter, but senior interviews reward judgment:

- when streams are expressive and efficient
- when loops are clearer and faster
- boxing costs and performance surprises
- collector characteristics and ordering/correctness pitfalls
- parallel stream caveats (short-circuit limits, work distribution)
