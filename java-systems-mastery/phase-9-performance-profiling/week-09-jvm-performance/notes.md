## Week 09 - JVM Performance and Profiling (Senior/Staff Stub)

This week builds production diagnosis intuition:

- JIT warmup and inlining (why benchmarks can mislead)
- escape analysis and allocation hotspots
- lock contention: what it looks like in CPU profiles and thread dumps
- reading flame graphs and mapping them back to code
- JFR/JMC basics and what runtime evidence to trust first

We keep this conceptual to avoid becoming “JVM tuning only”.
