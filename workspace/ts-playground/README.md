# TypeScript playground

Run with `ts-node` or `tsx` (no Angular). Use when you want types or a step between JS and Angular.

```bash
npm install
npx tsx example.ts
# or: npx ts-node example.ts
```

## Study rule (every topic)

1. Define in one line → [../mental-models.md](../mental-models.md)  
2. Flow in plain words → [../mental-models.md](../mental-models.md)
