## Example: Execution context

**Definition (one line):**  
The environment where a piece of code runs: variables, `this`, and the scope chain.

**Flow (plain words):**  
1. Script starts → Global EC is created (creation phase: allocate memory, hoist).  
2. Execution phase runs line by line; assignments happen.  
3. Function is called → new Function EC is created; when it returns, that EC is popped.
